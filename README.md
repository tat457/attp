### 2. 足踏みゲーム

- 上半身から足元まで、できるだけ体全体が映る位置に立ちます
- 先に `スタート` ボタンを押してから始めます
- 左右の膝を交互に持ち上げて足踏みします
- 足踏みが認識されるたびに全面雪原に足跡が増えます

### 3. ジャンプゲーム

- 足元までカメラに入る位置に立ちます
- 先に `スタート` ボタンを押してから始めます
- 軽くジャンプして踵を上下させます
- ジャンプが認識されると正面の階段を一段ずつ登っていきます
- ジャンプが認識されると、現在の段が大きく前面に表示されながら一段ずつ登っていきます

## 音について

- ブラウザ仕様のため、最初に画面を1回クリックまたはタップすると音が有効になります
- 進捗音、足踏み音、ジャンプ音、クリア音を入れています
- `スタート` ボタンを押すと `fitnes.mp3` が再生されます
- `fitnes.mp3` は `index.html` と同じフォルダに置いてください
- BGM に加えて、進捗音、足踏み音、ジャンプ音、クリア音を入れています
