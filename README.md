# ウォームアップゲーム3種

ブラウザで動く運動ゲームです。カメラ映像を使って、次の 3 モードを切り替えて遊べます。

- 1. 窓ふきゲーム: 手のひらの動きを検出し、50 回分の拭き動作で窓が全面きれいになります
- 2. 足踏みゲーム: 膝の上下動を検出し、50 回足踏みすると雪の上が足跡でいっぱいになります
- 3. ジャンプゲーム: 踵の上下動を検出し、50 回ジャンプすると階段の頂上に着きます

## 特徴

- `MediaPipe Holistic` を利用して、姿勢と手を 1 系統で検出
- 手のひら・膝・踵それぞれに合わせた判定ロジックを分離
- 平滑化を入れて、急なブレでカウントが暴れにくい構成
- ビルド不要の静的ファイル構成

## ファイル構成

- `index.html`: 画面全体の UI
- `styles.css`: レイアウトと見た目
- `app.js`: カメラ処理、検出、ゲームロジック

## 起動方法

カメラを使うため、`file://` ではなく `localhost` か `https` で開いてください。

### Python を使う場合

```bash
python3 -m http.server 8000
```

その後、ブラウザで次を開きます。

```text
http://localhost:8000
```

## 遊び方

### 1. 窓ふきゲーム

- カメラに手のひらが映るようにします
- 窓枠の中で手を大きく左右または円を描くように動かします
- 動いた量に応じて窓が少しずつきれいになります

### 2. 足踏みゲーム

- 上半身から足元まで、できるだけ体全体が映る位置に立ちます
- 左右の膝を交互に持ち上げて足踏みします
- 足踏みが認識されるたびに雪上に足跡が増えます

### 3. ジャンプゲーム

- 足元までカメラに入る位置に立ちます
- 軽くジャンプして踵を上下させます
- ジャンプが認識されると階段を一段ずつ登っていきます
