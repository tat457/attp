
# ウォームアップゲーム3種

ブラウザで動く運動ゲームです。カメラ映像を使って、次の 3 モードを切り替えて遊べます。
ブラウザで動く運動ゲームです。カメラで動きを検出しますが、画面にはカメラ映像を出さず、ゲーム画面だけを表示します。

- 1. 窓ふきゲーム: 手のひらの動きを検出し、50 回分の拭き動作で窓が全面きれいになります
- 2. 足踏みゲーム: 膝の上下動を検出し、50 回足踏みすると雪の上が足跡でいっぱいになります
- `MediaPipe Holistic` を利用して、姿勢と手を 1 系統で検出
- 手のひら・膝・踵それぞれに合わせた判定ロジックを分離
- 平滑化を入れて、急なブレでカウントが暴れにくい構成
- 効果音と分かりやすい `CLEAR!` 演出を搭載
- ビルド不要の静的ファイル構成

## ファイル構成
### 1. 窓ふきゲーム

- カメラに手のひらが映るようにします
- 窓枠の中で手を大きく左右または円を描くように動かします
- 画面の窓枠に合わせて、手を大きく左右または円を描くように動かします
- 動いた量に応じて窓が少しずつきれいになります

### 2. 足踏みゲーム

- 上半身から足元まで、できるだけ体全体が映る位置に立ちます
- 左右の膝を交互に持ち上げて足踏みします
- 足踏みが認識されるたびに雪上に足跡が増えます
- 足踏みが認識されるたびに全面雪原に足跡が増えます

### 3. ジャンプゲーム

- 足元までカメラに入る位置に立ちます
- 軽くジャンプして踵を上下させます
- ジャンプが認識されると階段を一段ずつ登っていきます
- ジャンプが認識されると正面の階段を一段ずつ登っていきます

## 音について

- ブラウザ仕様のため、最初に画面を1回クリックまたはタップすると音が有効になります
- 進捗音、足踏み音、ジャンプ音、クリア音を入れています
